1º Passo - Inicie a sua Venv > Venv\Scripts\activate

2º Passo - Instale todas as dependências que estará no arquivo requirements.txt > pip install -r requirements.txt

3º Rode a API e o | Abra 1 terminal e rode a aplicação com o comando : python manage.py runserver 

Nessa url você consegue fazer requisições GET e POST : http://127.0.0.1:8000/api/filtrar-veiculos/

Nessa Outra URL você consegue inserir alguns filtros : http://127.0.0.1:8000/automoveis/consulta-veiculos/
